# Kaede
TCP、UDP、HTTPのような一般的なプロトコルを扱うためのPythonライブラリ

## システム要件
- Linux / macOS (Windowsへの対応は未定です)
- CPython 3.12+
- OpenSSL 3.6+ or 4.0+ (LibreSSLなどの派生では未検証です)

## 対応プロトコル
- ICMP
- DNS
- UDS
- TCP
- UDP
- QUIC
- HTTP/1/2/3

## クレジット
多くのコードは[t3tra-dev/tcp-ip](https://github.com/t3tra-dev/tcp-ip/)を参考にしています。
